# Tic-Tac-Toe with AI 👾

A cross-platform console game featuring an unbeatable AI using the **minimax algorithm**.

## Features ✨
- Player vs. AI mode
- Dynamic scoreboard
- Cross-platform support (Windows/Linux/macOS)
